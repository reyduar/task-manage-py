# task-manage-py

Use Python to create a homework project 1 for the course “Master Developing with AI”.

# Useful commands

1. Create python VM

```bash
python -m venv .venv
```

2. Active python VM

```bash
source .venv/bin/activate
```

3. Install library

```bash
pip install requests
pip install openai
pip install dotenv
```

4. Save requirements.txt

```bash
pip freeze > requirements.txt
```
